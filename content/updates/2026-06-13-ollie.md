Two things happened this week.

First, the Fleet web app got its full CRUD build (phases 0–5): you can now run the
back office from the browser, not just the API or MCP. Alongside it, an equipment
maintenance log (CRUD across REST, MCP and the SPA), an Events page redesigned
into a live ops feed, a grouped scope-gated sidebar nav, and driver-to-equipment
cross-linking.

Second — and the reason there's a public repo to link at all — I did the
open-source launch prep: an AGPL-3.0 license, a SECURITY policy with private
vulnerability reporting, a Contributor Covenant and a CONTRIBUTING guide with DCO
sign-off, and CI guards (gitleaks secret-scanning and CodeQL) so I'm not shipping
credentials or obvious vulnerabilities.
